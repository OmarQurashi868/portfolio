Heist

Github repo

This is a WIP demo recreation of an N64 game called "Conker's bad fur day", specifically a local multiplayer mode called "Heist".

The game involvs 4 players (or teams) competing to steal the most money and store it in their vaults, using various weapons laid throughout the map.

The game was built on the Godot engine which uses the GDScript language that is based on Python, the engine is completely free and open-source. Models were created using Blender.

Implemented so far is character movement and spawning, team and score logic, various weapons, attacking and death (with animations!), all in a simple test map inspired by the original.
